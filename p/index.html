linux 终端美化

安装zsh

apt-get install zsh

安装oh-my-zsh

sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

替换zsh

# zsh替换bash
chsh -s `which zsh`

修改终端主题

vim ~/.zshrc
# 修改以下参数为
ZSH_THEME="agnoster"

这时出现乱码，需要安装字体

apt-get install fonts-powerline
